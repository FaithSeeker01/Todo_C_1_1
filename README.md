  例如：直径为3米，那么半径为1.5米的球体
	套入体积公式为：V球=4/3*3.14*1.5³≈14.13立方米（保留2位小数）
	扩展知识，球的表面积公式为S球=4πR²
编程实现：输入半径r，输出球体的体积值与表面积值。
